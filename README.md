# hider
the script can hide anything in image files<br>
it supports jpg and png file for cover and all type of file with no limit in size for secret file 
<h1>usage</h1>
to use this script  you can run it with python like this:

```
 $ python3 hider.py cover.jpg secret.txt --output output.jpg
```

# install dependencies

 ```
$ pip install -r requirements.txt
```
